# PurchaseKit Example App

This Rails application demonstrates the PurchaseKit integration for in-app purchases.

It contains working code for subscription management using the Pay gem, authentication, and authorization based on active subscriptions.

## Quick Start

To start the Rails server run:

```bash
bin/setup
bin/rails db:seed
bin/rails server
```

This will download dependencies, create and seed the database, and start the server.

Sign in with email/password: `user@example.com`/`password`.

## Requirements

* Ruby 3.2+
* SQLite
* Rails 8.1+
